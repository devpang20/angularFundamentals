## Module

### 컴포넌트, 파이프, 서비스 같은 앵귤러 앱의 주요 부분을 기능단위로 그룹핑 하게 해준다.
- 모든 앵귤러 앱은 단일 Root Module을 가짐
- 다수의 Feature Module을 가질 수 있음.
- 재사용, 외부 배포용으로 사용가능
